# 機種別収支推移グラフ機能の実装

## 目的
特定の機種を選択した際に、その機種の「稼働回数（または日付）」ごとの収支、および「累積収支」の推移を折れ線グラフで表示し、機種ごとの好不調の波を可視化する。

## 前提技術
- Vue 3 (Composition API)
- Vite
- Chart.js (`vue-chartjs`)

## 対象ファイル
- `src/components/analytics/MachineAnalytics.vue` (既存)
- `src/composables/useAnalytics.js` (既存)

## 実装要件

### 1. データ抽出ロジックの追加 (`useAnalytics.js`)
- 特定の機種名を引数に取り、その機種に該当する `filteredEntries` を日付の古い順にソートして返す関数（例: `getMachineTrend(machineName)`）を追加してください。
- 各エントリーには、その時点での「累積収支（cumulativeProfit）」を計算して付与してください。

### 2. UIの拡張 (`MachineAnalytics.vue`)
- 既存の機種別集計テーブルに対して、行をクリック可能（クリッカブル）にしてください。
- 選択された機種の状態（例: `selectedMachine`）を管理し、選択された機種のグラフを描画するエリア（テーブルの下部、またはモーダルダイアログ等）を追加してください。
- **グラフ仕様**:
  - `vue-chartjs` の Line chart を使用。
  - **X軸**: 稼働日、または稼働回数（1回目、2回目...）。
  - **Y軸（左・棒グラフ）**: その日の単発収支（プラスなら緑系、マイナスなら赤系）。
  - **Y軸（右・折れ線）**: 累積収支（全体のトレンドライン）。
  - 複合グラフ（Mixed Chart）として実装すると見やすくなります。

### 3. デザイン・スタイル
- グラフエリアが表示された際のスムーズなトランジション（CSSアニメーション）を追加するとUI体験が向上します。
- カラースキームは既存のダークテーマ（勝ち: `#22c55e`、負け: `#ef4444`、アクセント: `#00d4ff`等）に準拠してください。
